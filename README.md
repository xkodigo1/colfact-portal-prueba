# Colfact Portal

Segunda etapa del portal administrativo de Colfact con autenticación reactiva y rutas protegidas.

## Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- TanStack Query
- Axios
- React Hook Form + Zod
- MSW

## Scripts

```bash
npm install
npm run dev
npm run typecheck
npm run lint
npm run build
```
